# Eclipse main shortcuts

##General
- **<kbd>Ctrl</kbd> + 3  OR  <kbd>⌘</kbd> + 3** - Go to quick access search for available views, actions, wizards, menus and more
- **<kbd>Alt</kbd> + <kbd>⇧</kbd> + Q Q  OR  <kbd>⌥</kbd> + <kbd>⌘</kbd> + Q Q** - Show all available views and select one or more to open
- **<kbd>Ctrl</kbd> + F  OR  <kbd>⌘</kbd> + F** - Find/replace in the current file
- **<kbd>Ctrl</kbd> + H  OR  <kbd>⌘</kbd> + H** - Find/replace in current fi le, project, or workspace
