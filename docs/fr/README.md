# Cours Java 21 – Version Française

Ce dossier contiendra la traduction en français du cours **Java 21 Certification Course**.

La version originale (en anglais) se trouve dans :

- `../en/`

> ⚠️ Statut : traduction en cours.  
> Certains modules peuvent être manquants ou partiels.

## Structure prévue

- `module-00/` – Pré-requis, environnement, Eclipse
- `module-01/` – Bases du langage
- `module-02/` – Contrôle de flux
- `module-03/` – API de base (String, Math, Date/Time…)
- `module-04/` – POO, exceptions, generics
- `module-05/` – Programmation fonctionnelle et Streams
- `module-06/` – Collections
- `module-07/` – Threads et concurrence
- `module-08/` – Fichiers, I/O et NIO
- `module-09/` – Système de modules (JPMS)
